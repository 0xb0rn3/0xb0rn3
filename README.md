I'm a cybersecurity innovator who bridges the gap between offensive and defensive security through purposeful automation. My work focuses on creating tools that don't just solve problems—they transform how security professionals approach their craft. Every project is designed with three principles in mind: usability without compromise, scalability through elegant design, and impact that extends beyond the immediate use case.

My approach combines deep technical expertise with a human-centered perspective, ensuring that complex security operations become accessible and efficient. Whether developing reconnaissance frameworks, building automation tools, or crafting custom solutions, I believe in code that serves a greater purpose in the cybersecurity ecosystem.
